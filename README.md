# Project: AI-Driven Semiconductor Conductivity Modeling  
### Research Focus: High-Temperature Semiconductor Materials & Conductivity Prediction

UConn Data Science & Engineering | Semiconductor Materials Research

---

## Project Description

This project leverages Machine Learning techniques to model and predict electrical conductivity in semiconductor materials. By implementing models such as Random Forest, with planned extensions to XGBoost and Artificial Neural Networks (ANN), the system aims to uncover patterns between material properties and performance.

The research focuses on analyzing key physical characteristics such as lattice structures and bandgap energy to identify cost-effective, high-temperature alternatives to silicon. The goal is to improve predictive accuracy while contributing to the development of next-generation semiconductor materials.

## Key Technical Features

- **Machine Learning Modeling:** Implementation of Random Forest for initial conductivity prediction.
- **Model Expansion:** Future integration of XGBoost and Artificial Neural Networks for improved performance.
- **Feature Analysis:** Investigation of bandgap energy, lattice structures, and other material properties.
- **Optimization Focus:** Identifying affordable and heat-resistant alternatives to traditional silicon-based materials.
